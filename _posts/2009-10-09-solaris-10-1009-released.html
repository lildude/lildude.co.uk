Sun has quietly released Solaris 10 10/09 (aka update 8 ) and it is now available for download. As usual, the "What's New" doc has all the gory details of what's new in this release, but for the lazy, here's a quick summary of some of the more significant new features:

  • You can now install and boot from a large device up to 2TB
  • ZFS flash install support
  • User and group level ZFS quotas
  • ZFS cache devices - use those new SSD devices here
  • Several ZFS property enhancements
  • Improved ZFS ACL support
  • shadowAccount on LDAP Support in PAM
  • Turbo charged pkg installation and patching
  • Many driver enhancements
  • Freeware version upgrades, notably Samba has been upgraded to 3.0.35

Of course there are also the usual plethora of bug fixes included too and the new kernel patch for this release (14144[45]) should be out soon.

Whilst the release of this revision has been rather quiet so far, I suspect there will be a bit more fan fair about it at Oracle OpenWorld this weekend.
