Oracle Solaris 11 Express 2010.11 Support Repository update 1 (SRU1) is now available on the Oracle external support repository (https://pkg.oracle.com/solaris/support).

In order to access the support repository, you will need to have a valid software support contract and then register and obtain the key and certificate from https://pkg-register.oracle.com.

You can find more information on the external support repository in "Support Package Repositories Explained" on My Oracle Support.

Details of the changes included in SRU1 can be found in the corresponding README.
