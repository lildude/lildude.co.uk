SmugGal is a PHP Class that allows you to add a gallery of your SmugMug pictures, to any website.

SmugGal is intended to integrate into your website without the need for extensive styling. Accordingly, a very minimalist stylesheet is provided.

SmugGal also takes advantage of Thickbox (http://jquery.com/demo/thickbox/) functionality, if you wish. It gracefully degrades if Javascript is not enabled in the user's browser.

So far there are two variants available:

  1. Standalone SmugGal, for use on any website.
  2. As a Wordpress plugin, for integration into your Wordpress site.

The only difference between the two is the additional files supplied within the archives. If you downloaded the Standalone version, you won't have the Wordpress specific files, and vice-versa.

SmugGal uses the latest phpSmug wrapper class.

IMPORTANT:: IMPORTANT

In order for SmugGal to display your images, they need to be externally viewable/linkable from SmugMug. Please ensure "ext. links" within SmugMug is set to true for those galleries you wish to show on your website. If you don't, you won't see any pictures.

Features

  • SmugGal is easy to install, configure and setup. You can have your SmugMug images in a gallery format on your website in next to no time.
  • SmugGal attempts to integrate into your website with the least amount of effort by using very generic styling. Accordingly, only a small amount of additional CSS styling may be required in order to get SmugGal integrated perfectly into your site.
  • Thickbox (http://jquery.com/demo/thickbox/) functionality can be enabled or disabled, depending on your needs. If enabled, it gracefully degrades if Javascript is not enabled in the user's browser.
  • SmugGal displays "unlisted" galleries, without changing any settings in your SmugMug settings. If you don't want to show these galleries, you don't have to.
  • SmugGal displays your image descriptions as they appear in SmugMug.
